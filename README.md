## wdo linux (prev. web distro debian based <=1.0)

Linux distro from scratch with big possiblity.
### Fully customizable:
- pkg manager
- init system
- kernel and bootloader
- libc (not recommended)
- core utils
- everyting else

### By default:
- complied packages
- script inits
- linux 6.0 and grub 2.0
- glibc (just best)
- busybox
